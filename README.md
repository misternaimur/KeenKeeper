# KeenKeeper

Live link: https://keenkeeperlive.netlify.app/

## Key Points

- Personal relationship management app
- Track friend interactions: Call, Text, Video
- Set friendship goal days and next due date
- Timeline page with dropdown filter (All, Call, Text, Video)
- Stats page with pie chart and hover percentage
- Data stored in localStorage (key: keenkeeper_timeline)

## Tech Stack

- Next.js 16.2.4
- React 19.2.4
- React DOM 19.2.4
- Tailwind CSS 4
- DaisyUI 5.5.19
